How to Configure
  1. Copy the example file
    cp .env.example .env
  2. Get your IBM API Key
    Visit cloud.ibm.com/iam/apikeys and create a new API key.
  3. Get your watsonx.ai Project ID
    Open dataplatform.cloud.ibm.com, create or select a project, and copy the Project ID from the project settings.
  4. Edit your .env file
    IBM_API_KEY=your_key_here
IBM_PROJECT_ID=your_project_id_here
IBM_URL=https://us-south.ml.cloud.ibm.com
IBM_MODEL=ibm/granite-13b-instruct-v2
  5. Restart the application
    python app.py
